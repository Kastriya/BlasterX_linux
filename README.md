# BlasterX Linux 

A lightweight Linux application for real-time CPu temperature monitoring.

## Installation

```bash
git clone https://github.com/Kastriya/BlasterX_linux.git
cd BlasterX_linux
pip install -r requirements.txt
```

## Requirements

```bash
sudo apt install lm-sensors
```

## Run

```bash
python main.py
```

## Features

- Real-time CPU temperature monitoring
- Lightweight and fast
- Simple GUI
- Linux only

Built for Linux users.
